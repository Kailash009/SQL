PL/SQL:-
----------------------
>> It stands for Procedural/Structured Query Language.
>> It is used to perform logic in your query using if-else and loops.
-----------------------------------------------------------------------------------------------------------------
Variable:-
--------------------------
>> It is a memory location i.e. used to store a value.
>> In SQL to create a variable we use 'declare' keyword.
>> In Sql variable name must contain @ Symbol.
>> If you want to store any value into a variable then we use set keyword.
--------------------------------------------------------------------------------------------------------------
How to check the type of a variable in SQL:-
---------------------------------------------------------------
declare @a int,@b nvarchar(500)
set @a=10
set @b='Rohit'
SELECT sql_variant_property(@a,'BaseType') as BaseType
SELECT sql_variant_property(@b,'BaseType') as BaseType
----------------------------------------------------------------------------------------
For Example:-
----------------------------
declare @a int
set @a=100
print('a='+cast(@a as nvarchar))
---------------------------------------------------------------------------------------
wap to find the sum of two number.
--------------------------------------------------------
declare @a int,@b int,@res int
set @a=100
set @b=500
set @res=@a+@b
print('The Sum of '+cast(@a as nvarchar)+' and '+cast(@b as varchar)+' is = '+cast(@res as nvarchar))
