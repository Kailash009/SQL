b) Non-ANSI (Old Style Format):- (Where)

>> It is used = operator.
>> In equi join we can't use these operators  >,<,<=,>=,<>
>> It returns Matching Rows from the table.
Syntax:-
-------------------
select  <column-List> from table1 t1,table2 t2 where <Condition>

b) Non-Equi Join.
>> Other than = sign.
>> In it, we use following operators  -   > >= < <=
------------------------------------------------------------------------------------------------------------------
c) Self Join.
>> Joining a table itself.
>> No. of alias to create duplicate of single table.

SET OPERATOR:-
-------------------------------------
>> It is used to combine two or more tables.
Before going to perform set operator in your table we need to follow following condition which are as follows:
	a) No. of column should be same of both the table.
	b) Order of column should be same.
	c) Data type must be same.
----------------------------------------------------------------------------------------------------------------------------------------------------------------
Types of Set Operator:-
-------------------------------------------------------------
1. Union.    ->>> Without duplicate. (It can't give duplicate value.)
Syntax:-
-------------------------------------------
select * from <TN1>
union
select * from <TN2>
-------------------------------------------
2. UnionALL. -> With Duplicate. (it give duplicate value.)
Syntax:-
-----------------
select * from <TN1>
union all
select * from <TN2>
----------------------------------
3. Intersect.  ->> It returns common record from multiple tables.
Syntax:-
-----------------
select * from <TN1>
intersect
select * from <TN2>
-------------------------------------------------
4. Except.
>> It returns all the records from table1 which doesn't match with table2.
------------------------------------------------------------------------------------------------------------------
Syntax:-
-----------------
select * from <TN1>
except
select * from <TN2>
