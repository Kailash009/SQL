SQL SERVER:-
===============================
>> It stands for 'Structured Query Language'.
>> It is the product of Microsoft.
>> It is a platform dependent language. (It installed only Windows O.S.)
>> It is not Open Source.
>> It is not a case sensitive.
>> It provide both GUI (Graphical User Interface) and CUI (Command User Interface) base enviornment.
>> Sql is a language in which we can retrieve,update,insert,delete and perform many more operations very effective and efficient manner.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
How to Install SQL Server:-
-----------------------------------------------
you need to install two things:-
1. SQL SERVER. -> It is responsible to execute your Sql Query.
https://www.microsoft.com/en-in/sql-server/sql-server-downloads    ---->>> EXPRESS.
MixMode (Windows Authentication and Sql Server Authentication). -> Select Mixed Mode.
2. SSMS. (Sql Server Management Studio). => In which we can write our Sql Query.
https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver16
